Provides a set of classes to manage the string, image, font and cursor resources for the com.ibm.xtools.emf.ui.type plugin.

Messages and labels on GUI components that need to be localized (adapted for a specific region or language) should be placed in the messages.properties file in this package.

Classes in the com.ibm.xtools.emf.ui.type plugin should use the {@link com.ibm.xtools.emf.ui.type.internal.l10n.ResourceManager} class to access to the I18N resources that is needs.

@canBeSeenBy com.ibm.xtools.emf.ui.type.*
