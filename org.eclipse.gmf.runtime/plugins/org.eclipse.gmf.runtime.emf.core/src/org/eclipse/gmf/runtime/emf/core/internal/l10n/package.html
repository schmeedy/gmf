Localization resources for the MSL Core plug-in. Clients of the MSL should not use this package.

Package Specification

The {@link org.eclipse.gmf.runtime.emf.core.internal.l10n.ResourceManager} class Manages string and image resources for the MSL Core plug-in. Strings are maintained in the messages.properties file in this package; images if any are in the /icons directory at the root of the plug-in.

@canBeSeenBy org.eclipse.gmf.runtime.emf.core.*
