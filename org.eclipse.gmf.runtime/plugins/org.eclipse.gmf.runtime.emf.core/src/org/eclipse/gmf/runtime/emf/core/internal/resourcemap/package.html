Logical Resource physical mapping API.

Package Specification

Interface definitions for the physical mapping metamodel used by the logical resource implementation. A {@link org.eclipse.gmf.runtime.emf.core.internal.resourcemap.ResourceMap} is persisted as the first root of every physical resource in a logical resource, except for the case of a monolithic (single physical) resource.
