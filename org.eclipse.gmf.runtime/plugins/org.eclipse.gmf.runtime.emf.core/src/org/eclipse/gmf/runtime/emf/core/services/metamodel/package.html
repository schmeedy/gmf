Definition of the MSL meta-model support service provider API. MSL clients can customize various aspects of the MSL behaviour by providing a meta-model support implementation.

Package Specification

The meta-model service provider API consists of:

  • {@link org.eclipse.gmf.runtime.emf.core.services.metamodel.IMetamodelSupport}: specification of a meta-model. A meta-model provides information about structural relationships between metaclasses ({@link org.eclipse.emf.ecore.EClass}es), semantic procedures (similar to database triggers), etc.
  • {@link org.eclipse.gmf.runtime.emf.core.services.metamodel.IMetamodelSupportProvider}: interface for objects that map {@link org.eclipse.emf.ecore.EPackage}s to IMetamodelSupports. Providers are registered on the org.eclipse.gmf.runtime.emf.core.MetaModelProviders extension point
