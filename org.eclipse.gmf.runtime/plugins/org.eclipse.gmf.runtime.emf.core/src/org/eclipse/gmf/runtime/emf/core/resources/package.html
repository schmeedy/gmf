GMF resource API. This package provides the GMF resource factory responsible for creating GMF resources.

Package Specification

This package defines the {@link org.eclipse.gmf.runtime.emf.core.resources.GMFResourceFactory} class that creates GMF resources. It is not intended to be used by client code, but to be referenced on the EMF org.eclipse.emf.ecore.extension_parser extension point to associate the client application's file extension with the resource factory.
