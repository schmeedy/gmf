Diagram & Notation Domains: EditPart Service
§ An Editpart provider creates an EditPart for a given model
element.
§ Providers implement the IEditPartProvider with the following
method:
4 EditPart createGraphicEditPart(View view)
§ Providers can provide new editparts for new or existing model
element types.
§ Providers should avoid providing new editparts where the intent
is just to customize the set of editpolicies (this can be done more
efficiently using the EditPolicy service)
