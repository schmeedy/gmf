GMF Requirements Support: Runtime Framework
§
§
•Editor
•The runtime environment should support the following capabilities:
•Palette Extensible Palette Service
•Palette customizer [L] Standard GEF
•Sticky tool [M] Standard GEF
•Properties Leverages enhanced properties view submitted to Eclipse core
•Overview (bird's eye view) [H] Standard GEF
•Zoom +/- [M]
•Navigator [M] Not part of the initial submission, currently migrating to Common Navigator
•Outline [H] Standard GEF
•Decorators [M] Decorator Service
•Keyboard bindings [L] Can support accelerators, but not integrated with Eclipse key binding support
•Direct editing [H]
•Text validation/parser plug-in Parser service
•Marker support Navigation service
•Drag and drop [H] D&D service
