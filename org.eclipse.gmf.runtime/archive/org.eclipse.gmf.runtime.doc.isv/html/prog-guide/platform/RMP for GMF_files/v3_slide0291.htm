Common Domain: Marker Navigation Service
§ This service is the distribution point for "gotoMarker" requests made
by Eclipse to an IEditorPart.
§ In xtools editors, the implementation of the gotoMarker() method
simply forwards the request to this service.
§ Allows navigation to specific elements in diagrams.
4 Example: problem marker for an element, double click in the problems
view goes to the highlighted element in the diagram.
