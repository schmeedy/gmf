EMF & OCL Domains: Model Management Services (cont.)
§Meta-model Support Service
4The service is useful for supporting semantic procedures and post processing after loading resources. The methods to implement in the IMetamodelSupport interface are handleEvent() and postProcess().
4Extension point org.eclipse.gmf.runtime.emf.core. MetaModelProviders defined for clients to provide meta-model support services. Providers implement the interface IMetamodelSupportProvider. The implementation of the interface method getMetamodelSupport returns the IMetamodelSupport object.
4The service gives all interested providers a chance to create meta-model support objects for the given EPackage in order of priority from lowest to highest.
4Example
•Apply profiles and add required reference libraries for UML2 models as part of the post processing capability
•Implement a semantic procedure to auto-name books and writers when created within the library model
