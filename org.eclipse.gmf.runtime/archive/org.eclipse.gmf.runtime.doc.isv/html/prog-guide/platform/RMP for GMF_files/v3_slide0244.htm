Diagram & Notation Domains: EditPolicy Service
§ An Editpolicy provider creates EditPolicies for a given editpart.
§ Providers implement the IEditPolicyProvider with the following method:
4 void createEditPolicies(EditPart editpart)
§ The service gives all interested providers a chance to create editpolicies for
the given editpart in order of priority from lowest to highest.
§ Overriding an editpolicy is done by assigning a new one in the same role.
§ Assigning ‘null’ to an editpolicy role is equivalent to removing that role
behavior.
§ The service is very useful for adding/removing behavior without having to
subclass targeted editparts.
