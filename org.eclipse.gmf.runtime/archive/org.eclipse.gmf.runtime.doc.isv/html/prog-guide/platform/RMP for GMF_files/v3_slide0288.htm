Common Domain: Action Filter Service
§ A service that extends what Eclipse provides for plug-in action
enablement and visibility.
§ Provides a set of classes to help in evaluating expressions.
§ Allows clients to extend the action filters supported by objects
selected in workbench.
§ It also permits the entire workbench selection to be considered so
that an action can be enabled if it is supported on one or more
elements in the selection.
4 Example: An action filter that will enable an action if the selected element
can be duplicated.
4 Example: An action filter for elements that have a specific UML Stereotype
applied.
