Common Domain: Common utility classes
§ The common domain provides shared utilities used in all xtools plug-
ins.
4 AbstractResourceManager - I18N and resource management.
4 Log - A utility for logging errors, warnings, and information.
4 Trace - A utility for tracing debug information.
4 FileChangeManager – Handle resource change events.
4 FileModificationValidator – Determine if files are writable, check-out if
required.
4 Shared widgets, dialogs, preference page utilities, etc.
