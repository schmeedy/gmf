Rational Modeling Platform Basics
Set of frameworks specialized in the creation of Eclipse-based modeling features
Initially created for the internal usage of IBM teams and their partners
Leverages proven open source technologies: Eclipse Platform, EMF, GEF
V1 introduced in the UML Visualizer feature of WSS AD 5.0 (A.k.a “Aurora”)
…no public API provided
V2 deployed in various Rational Software Development Platform products
…API was made public to support UML Modeling customers
V3, targeted for Eclipse 3.2 timeframe, is currently being developed and refactored to have broader APIs and cleaner dependencies
…this is what we propose to contribute to the GMF project
