Diagram & Notation Domains: Palette Service
§ Palette providers contribute to the palette of a given editor with a given content
§ Providers implement the IPaletteProvider with the following method:
4 void contributeToPalette(IEditorPart editor, Object content, PaletteRoot root)
§ Palette providers are encouraged to subclass the DefaultPaletteProvider, since all the
contributions could be described in the XML extension.
§ Palette providers can contribute:
4 New Palette Drawers
4 New Palette Groups
4 New Palette Separators
4 New Palette Tool Items in drawers/groups.
4 New Palette Template Items in drawers/groups
