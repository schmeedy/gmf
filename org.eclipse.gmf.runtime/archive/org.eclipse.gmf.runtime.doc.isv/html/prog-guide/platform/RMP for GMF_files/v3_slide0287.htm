Common Domain: Global Action Handler Service
§ This service provides an extensible way to handle commonly used
(global) actions in different views and editors.
§ These global actions may appear in a view part context menu or
may be predefined Eclipse retargetable actions.
§ The service allows different handlers to be used depending on the
nature of the element(s) selected in the workbench part.
4  Example: Override the global delete action when selecting an element on
a diagram.
