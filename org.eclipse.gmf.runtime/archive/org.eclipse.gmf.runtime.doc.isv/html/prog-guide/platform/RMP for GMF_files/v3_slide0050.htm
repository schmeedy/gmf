Diagram Domain
Enables the creation of a GEF-based diagram editor
• Provides an EMF-based notation meta-model
• Optimized for team support
• Enables separation of semantic and notation models (but not enforced)
• Provides standard actions (e.g. Z-Order commands, fonts, etc…)
• Provides standard contribution items (font, arrange all, layout, etc…)
• Provides clipboard support (SVG, etc…)
• Provides linked diagram support
• Reusable edit parts, edit policies and figures
• Provides basic properties (font, location, color, etc…)
• Reusable layouts
• Basic print and print preview support
