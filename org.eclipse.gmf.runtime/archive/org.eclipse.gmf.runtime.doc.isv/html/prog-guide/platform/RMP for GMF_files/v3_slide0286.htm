Common Domain: Parser Service
§ A service which provides parsed strings.
§ The service is given an operation with a source object and parser
options.
§ Based on the object and the provided options, a parser provider will
return a string.
4 Example: provide a name for the logic element.
4 Example: provide the fully qualified name for a UML Element.
4 Example: provide the edit string for a UML Operation contains the full
signature.
