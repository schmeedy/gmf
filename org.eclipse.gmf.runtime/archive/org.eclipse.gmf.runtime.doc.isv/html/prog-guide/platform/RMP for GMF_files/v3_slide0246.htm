Diagram & Notation Domains: Decoration Service
§ Decoration providers creates decorations for a given adapter.
§ Providers implement the IDecorationProvider with the following method:
4 void createDecorators(IDecoratorTarget decoratorTarget)
§ The IDecoratorTarget API lets the client install a custom Decoration or an
image to a pre-defined location on the shape.
§ The service could be used by any client that needs to decorate a shape on
the diagram without having any dependency to GEF.
