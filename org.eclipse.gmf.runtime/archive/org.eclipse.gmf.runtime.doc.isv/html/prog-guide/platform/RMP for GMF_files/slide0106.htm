GMF Requirements Support: Diagram Definition (Cont.)
§
§
•Notation Definition Outside the scope of this submission
•An editing environment for diagram notation definition will be provided, to specify:
•Nodes [H]
•Shape definitions, including geometric, custom, and image file.
•Links [H]
•Connector elements to indicate relationship/flow/etc.
•Decorators [M]
•Text and icons for Node and Link elements.
•Properties [H]
•Attributes of an element to be displayed in a properties view.
•A graphical surface for diagram definitions will be provided (bootstrapped) [M].
•Provide a WYSIWYG capability for diagram definition. [L]
•A graphical surface for mapping definitions will be provided (bootstrapped) [L].
•Tooling Definition Outside the scope of this submission
•Along with diagram definition and domain model mapping, the designer should allow for the definition of those runtime options supported (as listed below), such as palette configuration, overview support, etc. [H]
