Diagram & Notation Domains: Benefits
§ Provides a logical persistence model for GEF through the
presentation layer.
• Defines a notation meta-model in EMF that is generic and logically
separated from the semantic model persistence.  Clients of RMP-DL
have diagram interoperability and compatible formats.
• Allows utilization of additional services developed by RMP (common /
MSL – undo / redo, query, copy/paste etc.)
• Reuse of diagram code in different contexts.  Semantic model could be
implemented through a bridge pattern.
§ Provide an extensibility mechanism for GEF concepts.
• Support for on-demand loading for diagram components
• Ease of maintenance through component based architecture.
• Clients inherit extensibility mechanism that can be harnessed for their
own individual domain extensions through partners.
