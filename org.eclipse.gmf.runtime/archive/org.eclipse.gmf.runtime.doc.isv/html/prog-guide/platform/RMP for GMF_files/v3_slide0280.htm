Common Domain: Command infrastructure
§ Provides a set of classes for executing, undoing and redoing units of
work, known as Commands .
§ This API is an implementation of the Command design pattern.
§ Using this API, application operations can be done, undone, redone
and canceled without any dependency on UI components.
§ A command manager will maintain a history of the commands that it
has executed that are eligible to be undone and redone.
§ Is currently independent of Eclipse but will be integrated with the
Eclipse operations framework that is coming in Eclipse 3.1.
