•Editor (continued)
•The runtime environment should support the following capabilities:  (continued)
•Layout [L]
•A number of layout strategies may be appropriate for a particular diagram and the runtime framework should provide for this and for the addition of layout plug-ins to be added via extension point. Layout Service
•Alignment [L]
•Align selected nodes to top, bottom, right, left, center. Supported and productized
•Set width, height of elements. Supported and productized through properties view
•Support standard graphical editor facilities (actions, rulers, guides) provided by GEF. [L] Standard GEF
•Provide support for compartments/subcompartments. [M] Supported and productized optionally collapsible compartments
•Feedback (in status line) for constraint violation, advice, etc. [M] Constraint failure are shown in problems view
•Filter views [L]
•Filter by element type, property, or regular expression. Not regular expressions
•Hide/Show individual elements. Supported in canonical views
