Common Domain: Contribution Item Service
§ Provides a set of classes that facilitates the contribution of actions to
Eclipse workbench parts.
§ It is meant to address some of the deficiencies in the Eclipse
support for static action contributions, as follows:
4 Adds support for static contribution of custom contribution items
4 Adds support for static menu contributions to toolbar managers
4 Adds support for static contributions to menu managers that have been
added programmatically to a toolbar
