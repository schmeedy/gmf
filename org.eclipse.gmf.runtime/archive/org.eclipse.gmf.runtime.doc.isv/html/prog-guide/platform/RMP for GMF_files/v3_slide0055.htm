EMF & OCL Domains
The EMF Domain provides useful infrastructure for any EMF-based applications:
• Model Management Services
• Batched notifications support
• Multi-threading access protocol
• Object life-cycle support
• Model Integrity Services
• OCL and Java Constraints Support
• Model Query Services
• OCL and Java based Queries
• Model Clipboard Services
• Extensible Type Registry
• Index Service
