Common Domain: Properties Service
§ A set of services that allow clients to extends properties provided by
a domain model.
§ An EMF model has a defined set of properties, clients wish to
extend these properties.
§ Property providers allows a client to add additional properties to the
domain.
4 Example: Allow a customer to add custom properties.
§ Property modifiers modify the domain properties.
4 Example: Override the default property modifiers to make them read only.
