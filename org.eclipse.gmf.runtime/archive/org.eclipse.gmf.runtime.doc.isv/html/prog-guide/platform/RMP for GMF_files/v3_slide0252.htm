Example: Creating a Logic diagram editor (Cont.)
§ 5. Creating the controller (EditPart) for the shape.
4 Create a EditPart Provider class that will create the editpart controller for
the shape  (LogicEditPartProvider)
4 In the EditPart provider class, override the getNodeEditPartClass method
to return the EditPart class name based on the View type passed into the
method. (i.e. LEDEditPart)
4 Define the EditPart extension point and add the on-demand loading
conditions.
§ 6. Displaying the shape on the diagram surface.
4 Create a new draw2d IFigure subclass that overrides the paintFigure
routine to draw the figure as desired. (LEDFigure)
4 In the EditPart created in step 4, override the createNodeFigure routine to
return the new Figure you just created..
