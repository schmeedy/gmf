Common Domain: Services
§Each service performs some specific piece of work for clients.
§An operation represents the functionality provided by a service.
§Operations are executed by passing them a provider to do the real work, based on the provider’s interface.
§The Service uses execution strategies to select providers to service a client request.
4FORWARD: The operation is executed by all providers in forward order.
4FIRST: The operation is executed by the first provider.
§Services also have performance optimization settings (caching).
§Each service provides an Eclipse extension point so that providers can register with the service:
