  1. Concepts

 

Contribution – extension, new feature.

Contribution item – contribution to a shared UI resource such as a menu or toolbar.  All contributions are managed by some contribution manager.  For example, button contributed to a tool bar is referred to as a ‘contribution item’.  Menu contributed to a menu bar is refered to as a contribution item.  Menu item contributed to a menu is referred to as a contribution item.

Contribution manager – organizes contribution to UI components, keeps track of a list of contribution items.  You can talk to a contribution manager using IContributionManager interface.  Through ContributionManager you can add new contribution, remove a contribution, retrieve a contribution, contribute new action.

 

Also, please review the following API documentation:

  • IContributionManager
  • ContributionManager
  • MenuManager
  • ToolBarManager
  • StatusLineManager
