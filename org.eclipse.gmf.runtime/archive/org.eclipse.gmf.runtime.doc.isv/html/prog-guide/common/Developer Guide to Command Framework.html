  1. Introduction

 

This document describes how to use Aurora Command Framework.  Aurora commands can be divided into two types:

  1. Regular commands – ex. copy image, select, open diagram.
  2. Commands that interact with the model – ex. create, delete, duplicate, copy, cut, paste.

  1. Use Cases

 

6.1  Contribute new executable/undoable/redoable command

6.1.1        Extend org.eclipse.gmf.runtime.common.core.command.AbstractCommand

6.1.2        Overwrite doExecute() (here you put the logic of your command)

6.1.3        Implement doUndo().  Use return newOKCommandResult() to return CommandResult object.

6.1.4        Implement doRedo().  Use return newOKCommandResult() to return CommandResult object. 

6.1.5        Overwrite isUndoable() to return true

6.1.6        Overwrite isRedoable() to return true

6.1.7        Code sample

 

import org.eclipse.core.runtime.IProgressMonitor;

import org.eclipse.gmf.runtime.common.core.command.AbstractCommand;

import org.eclipse.gmf.runtime.common.core.command.CommandResult;

 

/**

 * Demonstrates the usage of command framework.

 */

public class SampleIncreaseCommand extends AbstractCommand {

 

      private int fValue = 0;

      private int fLastValue = -1;

     

      /**

       * @param label

       */

      public SampleIncreaseCommand(String label) {

            super(label);

            System.out.println("Created command " + label);

      }

 

      // The logic of your command

      protected CommandResult doExecute(IProgressMonitor progressMonitor) {

            fLastValue = 10;

            increase(fLastValue);

            System.out.println("Value after execute: " + fValue);

            return newOKCommandResult();

      }

     

      // The redo logic.

      protected CommandResult doRedo() {

            increase(fLastValue);

            System.out.println("Value after redo: " + fValue);

            return newOKCommandResult();

      }

     

      // Undo logic.

      protected CommandResult doUndo() {

            decrease(fLastValue);

            System.out.println("Value after undo: " + fValue);

            return newOKCommandResult();

      }

     

      // This method must return ‘true’ in order for your command to be redoable.

      public boolean isRedoable() {

            return true;

      }

     

      // This method must return ‘true’ in order for your command to be undoable.

      public boolean isUndoable() {

            return true;

      }

     

      // This method increases the value field by specified value.

      private void increase(int value) {

            fValue += value;

      }

     

// This method decreases the value of field by specified value.

      private void decrease(int value) {

            fValue -= value;

      }

}

6.10.3    Code sample.

…

public class MyListener implements ICommandManagerChangeListener {

public void commandManagerChanged(CommandManagerChangeEvent event) {

      // some event processing logic

}

}

…

// Instantiate listener

MyListener myListener = new MyListener();

// Register listener with CommandManager

cmdMgr.addCommandManagerChangeListener(myListener);
